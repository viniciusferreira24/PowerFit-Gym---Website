# 💪 PowerFit Gym — Website

Website profissional de academia desenvolvido como parte de um portfólio de projetos de **desenvolvimento web front-end**.
O objetivo deste projeto é demonstrar a criação de um **site moderno, responsivo e visualmente atrativo**, simulando um site real para uma academia.

---

# 🚀 Demonstração do Projeto

Este projeto representa o site institucional de uma academia fictícia chamada **PowerFit Gym**, com foco em:

* apresentação da academia
* exibição de planos
* treinos da semana
* galeria de estrutura
* contato direto via WhatsApp

O layout foi pensado para ser **moderno, dinâmico e focado em conversão**, simulando um site que poderia ser utilizado por uma academia real.

---

# 🎯 Objetivo do Projeto

Este site foi desenvolvido para:

* compor um **portfólio profissional de desenvolvimento web**
* praticar **HTML, CSS e JavaScript**
* desenvolver **layouts responsivos**
* aplicar **animações e interações modernas**

O projeto simula um **site real para clientes do setor fitness**.

---

# 🧰 Tecnologias Utilizadas

O projeto foi desenvolvido utilizando tecnologias fundamentais do desenvolvimento web:

* **HTML5** — estrutura do site
* **CSS3** — estilização e layout
* **JavaScript (Vanilla JS)** — interações e animações
* **Google Fonts** — tipografia moderna

# ✨ Funcionalidades do Site

O site inclui diversas funcionalidades comuns em sites profissionais:

### 🎨 Interface Moderna

* design escuro moderno
* tipografia limpa
* layout organizado

### 🖱️ Interações Dinâmicas

* animação ao passar o mouse nos cards
* efeito de zoom nas imagens

### 📊 Contadores Animados

Indicadores animados mostrando dados da academia:

* alunos ativos
* treinadores

### 🎬 Animações ao Rolar a Página

Elementos aparecem suavemente conforme o usuário rola o site.

### 📱 Design Responsivo

O layout se adapta a:

* celulares
* tablets
* computadores

### 💬 Integração com WhatsApp

Botão flutuante que abre diretamente uma conversa no WhatsApp.
